PROJECT SCOPE

This project is about the creation of a user friendly software application to help Individuals discover favourable apartments for rent, lease or even acquisition from wherever they are. This software also comes with a unique feature that allows users tour their potential apartment

Although initiated to make property search and acquisition seamless in Nigeria, It is intended to widen its reach in due time.
